gradle
==========

## install  
```  
$wget https://services.gradle.org/distributions/gradle-2.9-all.zip
$unzip gradle-2.9-all.zip
$set GRADLE_HOME
$export PATH="$PATH:$GRADLE_HOME/bin"
$gradle -v
```  
> 如果执行`gradle -v`提示“Pessmion denied”，为gradle文件赋可执行文件的权限。
